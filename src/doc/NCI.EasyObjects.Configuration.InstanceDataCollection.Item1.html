InstanceDataCollection.Item Property (Int32)

Gets or sets the InstanceData at the specified index.

public InstanceData this[
   int index
] {get; set;}

Parameters

index

The index of the InstanceData to get or set.

Property Value

The value associated with the specified index. If the specified index is not found, attempting to get it returns a a null reference reference (Nothing in Visual Basic), and attempting to set it creates a new entry using the specified index.
