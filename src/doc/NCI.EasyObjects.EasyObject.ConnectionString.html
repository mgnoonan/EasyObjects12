EasyObject.ConnectionString Property

Gets or sets the connection string used to connect to the database.

public string ConnectionString {get; set;}

Remarks

This will override any settings in dataConfiguration.config and instead connect directly using the connection string. Leave this field blank to use database instances from dataConfiguration.config.

You can control the connection settings through the UseIntegratedSecurity, ConnectionUserID and the ConnectionPassword properties. credentials.
