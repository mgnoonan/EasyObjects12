EasyObjects.NET Programmers Reference

DynamicQuery.AddOrderBy Method (String, Dir)

Adds an ORDER BY clause to the query using the FieldName. If you want to order the data by two columns you will need to call this method twice.

public virtual void AddOrderBy(
   string fieldName,
   Dir direction
);

Parameters

fieldName
A column to sort by
direction
A WhereParameter.Dir indicating the sort direction

Example

VB.NET 

emps.Query.AddOrderBy("ExampleColumn", WhereParameter.Dir.DESC)
emps.Query.AddOrderBy(EmployeesSchema.LastName, WhereParameter.Dir.ASC)
emps.Query.AddOrderBy(EmployeesSchema.FirstName, WhereParameter.Dir.ASC)
C# 
emps.Query.AddOrderBy("ExampleColumn", WhereParameter.Dir.DESC);
emps.Query.AddOrderBy(EmployeesSchema.LastName, WhereParameter.Dir.ASC);
emps.Query.AddOrderBy(EmployeesSchema.FirstName, WhereParameter.Dir.ASC);
